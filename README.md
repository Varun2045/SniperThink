# SniperThink - Strategic Intelligence Platform

A cutting-edge, AI-powered strategic intelligence platform that helps businesses achieve market dominance through precision strategy, flawless execution, and continuous optimization.

## 🚀 Features

### Core Functionality
- **5-Step Strategic Process**: Analyze → Strategize → Execute → Optimize → Dominate
- **Interactive Strategy Steps**: Each step with detailed features, animations, and call-to-actions
- **Lead Generation**: Professional inline forms with smooth transitions
- **Responsive Design**: Mobile-first approach with seamless desktop experience
- **Dark Mode Support**: Full theme compatibility throughout the application

### Technical Highlights
- **Modern React 18**: Hooks, functional components, and state management
- **Framer Motion**: Advanced animations and micro-interactions
- **Lucide Icons**: Professional, scalable icon system
- **Tailwind CSS**: Utility-first styling with custom gradients
- **Axios Integration**: API connectivity for lead generation
- **Component Architecture**: Modular, reusable React components

### User Experience
- **Smooth Scrolling**: Navigation between sections with scroll progress
- **Interactive Elements**: Hover states, loading animations, and transitions
- **Glass Morphism**: Modern UI with backdrop blur and transparency
- **Professional Animations**: Spring physics and natural motion design
- **Form Validation**: Real-time feedback and error handling

## 🛠️ Technology Stack

### Frontend
- **React 18.2.0** - UI framework with hooks

## 🚀 Getting Started

### Prerequisites
- Node.js 16+ 
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd project-2/frontend
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Environment Setup**
   ```bash
   cp .env.example .env
   # Add your backend URL to VITE_BACKEND_URL
   ```

4. **Start Development Server**
   ```bash
   npm run dev
   ```

5. **Build for Production**
   ```bash
   npm run build
   ```
